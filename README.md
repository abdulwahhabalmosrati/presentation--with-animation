## How to Use

### Local Development
1. Clone the repository
2. Start a local server:
   ```bash
   python3 -m http.server 8000
   # or
   npx serve .
   # or
   php -S localhost:8000
   ```
3. Open `http://localhost:8000` in your browser

### Direct Usage
Simply open `index.html` in any modern web browser.
